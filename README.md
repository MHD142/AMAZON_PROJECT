# Amazon Product Page Clone - Water Bottle

This project is a front-end practice project where I recreated an Amazon-style product page for a water bottle using only HTML and CSS.

The goal of this project was to practice building a realistic product page layout, including the header, product image section, product information, price area, quantity selection, buttons, and footer-like layout.

## Live Demo

You can view the deployed project here:

[Live Website](https://moeameramazonproject.netlify.app/)

## Project Features

- Amazon-style product page layout
- Product image section
- Product details and description
- Price and delivery information
- Quantity selection area
- Add to cart / buy button styling
- Responsive layout adjustments using `@media`
- Hover and focus effects using CSS
- Built without JavaScript

## Project Purpose

This project was created as part of my practice in HTML and CSS.  
I wanted to improve my understanding of page structure, layout design, responsive styling, spacing, positioning, and CSS selectors.

## What I Practiced

In this project, I practiced:

- Semantic HTML structure
- CSS Flexbox
- CSS positioning
- Media queries
- Responsive units such as `rem`, `vw`, and `vh`
- Hover effects
- Form elements such as `select`, `option`, and checkbox inputs
- Styling product cards and product information sections
